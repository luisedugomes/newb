# 🎨 Design System - ArenaRank

Neste projeto, utilizamos um framework UI e aplicamos customizações pontuais para refletir a identidade visual.

### 1. Framework Base

- **Framework escolhido:** Bootstrap v5.3
- **Motivação:** Oferece componentes HTML e CSS fornecendo uma rápida prototipação de projeto e um código fácil de se ler e implementar melhorias ao decorrer do projeto;

### 2. Palheta de cores:

As variáveis de cor do framework foram escolhidas para induzir o usuário se sentir em um domínio gamer esportivo moderno, para que ele indentifique logo de cara o objetivo da plataforma:

- Neon Cyan `#0de3f2` 
- Void Black `#050507` 
- Carbon Surface `#121216` 
-  Ice White `#E0E0E6` 
-  Live Red `#FF003C` 
-  Tactical Purple `#B026FF`
-  Grayish Blue `#849395`
-  Dark Bluish Gray `#4a4a5a`
-  Cool Charcoal Gray `#1a1a20`

### 3. Tipografia

- **Títulos e Textos:** `Space Grotesk`
